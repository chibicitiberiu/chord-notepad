===============
Quick Reference
===============

This page contains complete reference tables for all chord types, directives,
and notation supported by Chord Notepad.

Directive Reference
===================

All Directives
--------------

.. list-table::
   :header-rows: 1
   :widths: 20 35 45

   * - Directive
     - Purpose
     - Syntax
   * - {bpm}
     - Set or change tempo
     - See below
   * - {tempo}
     - Same as bpm
     - Same as bpm
   * - {time}
     - Set time signature
     - ``{time: beats/unit}``
   * - {key}
     - Set musical key
     - ``{key: note}``
   * - {label}
     - Mark a position
     - ``{label: name}``
   * - {loop}
     - Repeat a section
     - ``{loop: name count}``

BPM Directive Values
--------------------

.. list-table::
   :header-rows: 1
   :widths: 30 30 40

   * - Type
     - Format
     - Example
   * - Absolute
     - Number (60-240)
     - ``{bpm: 120}``
   * - Relative add
     - +number
     - ``{bpm: +20}``
   * - Relative subtract
     - -number
     - ``{bpm: -15}``
   * - Percentage
     - number%
     - ``{bpm: 50%}``
   * - Multiplier
     - numberx
     - ``{bpm: 2x}``
   * - Reset
     - reset or original
     - ``{bpm: reset}``


Duration Reference
==================

.. list-table::
   :header-rows: 1
   :widths: 30 30 40

   * - Duration
     - Beats
     - Example
   * - (default)
     - 1
     - ``C`` = 1 beat
   * - \*2
     - 2
     - ``C*2`` = 2 beats
   * - \*4
     - 4
     - ``Am*4`` = 4 beats
   * - \*0.5
     - 0.5
     - ``G*0.5`` = half beat
   * - \*1.5
     - 1.5
     - ``F*1.5`` = 1.5 beats


Special Syntax
==============

.. list-table::
   :header-rows: 1
   :widths: 25 35 40

   * - Syntax
     - Purpose
     - Example
   * - ``//``
     - Comment (to end of line)
     - ``C Am // verse chords``
   * - ``/``
     - Slash chord (bass note)
     - ``C/G``, ``Am/E``
   * - ``*``
     - Duration modifier
     - ``C*2``, ``Am*4``
   * - ``{}``
     - Directive
     - ``{bpm: 120}``
